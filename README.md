# Golden Crust Bakery

A simple bakery website built with HTML, CSS, JavaScript, PHP, and MySQL.

## Features
- Responsive homepage
- Menu section
- About section
- Seat booking form
- Contact form
- Admin panel

## How to Run
1. Put the project folder in `htdocs`.
2. Start Apache and MySQL in XAMPP.
3. Import or create the required tables.
4. Open:
   `http://localhost/golden-crust-bakery/index.php`

## Files
- `index.php`
- `style.css`
- `script.js`
- `config.php`
- `booking.php`
- `contact.php`
