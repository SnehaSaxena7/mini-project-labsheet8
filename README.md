#  BookVault - PHP Bookstore

##  How to Run
1. Install XAMPP / WAMP / MAMP
2. Move project to htdocs/bookvault
3. Open http://localhost/bookvault/

## Features
- Browse books
- Search & filter
- Cart & checkout
- Wishlist
- User account system
- No database required (PHP sessions)
